## Dependencies

1. Compiles only with **XeTeX** and required **BibTex** for compiling publications and the .bib filetype.
2. Uses open fonts and logo images in repo.

## Usage

Generate .pdf with xelatex:
```
xelatex tstesco_resume.xtx
```
